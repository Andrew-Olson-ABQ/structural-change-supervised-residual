Structural Change Detection Using Supervised Learning Residuals

Graduate Course Project — STAT 5820
Andrew Olson | MA in Applied Mathematics, Wayne State University


Overview

This project demonstrates how supervised learning models can be used to detect structural change in a time series, by training a model on data before a known change point and examining how its prediction residuals behave after the change.

The core idea: if a model is trained only on the "normal" relationship between predictors and a response, its prediction errors (residuals) should grow noticeably once the underlying relationship shifts. Tracking residuals over time turns any supervised model into an implicit change-point detector.


Methodology

1. Learn a baseline relationship from real data
A linear regression model was fit on R's built-in airquality dataset (Ozone ~ Solar.R + Wind + Temp + Month + Day) to obtain a realistic baseline function f(X).

2. Generate a time series with a known, injected structural change
Using f(X), a synthetic time series of 2,000 observations was generated by resampling predictors from the real data:


Before the change point (t ≤ 1000): Y = f(X) + noise
After the change point (t > 1000): Y = f(X) + c · (Temp_standardized)² + noise


The added term introduces a nonlinear structural shift in how Temp relates to the response, simulating a real-world regime change. The change magnitude c (default = 15) was also varied (5, 15, 25) in a sensitivity analysis.

3. Train models only on the pre-change data
Both a linear regression model and a random forest model were trained exclusively on the unchanged portion of the data (t ≤ 1000), then used to predict across the entire time series, including the post-change period they never saw.

4. Detect the change via residual analysis
Absolute residuals were tracked over time for both models. A clear, statistically significant increase in residuals after t = 1000 confirms the models are detecting the structural break, even without being told where it occurs.


Key Results


Both models show a sharp, visually obvious increase in absolute residuals immediately after the true change point (t = 1000)
A two-sample t-test comparing pre- and post-change residuals confirmed the increase is statistically significant
Sensitivity analysis: larger injected change sizes (c = 5, 15, 25) produced proportionally larger increases in residuals for both models, confirming the method scales with the strength of the underlying structural shift
Random forest and linear regression showed similar overall sensitivity to the change, despite differing model complexity

Analysis was performed in R. Key packages used:


ggplot2 — visualization

How to Run

Run STAT5820_project_analysis.R top to bottom. The script will:


Load and clean the airquality dataset
Fit the baseline regression model
Generate the synthetic change-point dataset
Train both models on the pre-change data
Produce all residual diagnostic plots (saved to figures/)
Run the sensitivity analysis across multiple change sizes
